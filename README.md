# ☣️ Quarantine Lab: L.A.E.S. V2

> Tactical Labrador-Sized Animal Encounter Simulator

Quarantine Lab (L.A.E.S. V2) is a single-page AI-powered survival simulator where ordinary creatures become Labrador-sized biological threats.

Choose an environment, equipment, and behavioral stance, then attempt to survive dynamic encounters generated by AI or the built-in simulation engine.

Built as a standalone HTML application with zero installation required.

---

# Features

## 🧬 Creature Synthesizer

Create encounters using:

- Built-in hostile species
- Custom creatures
- Dynamic threat generation
- Procedural statistics

Every creature has unique:

- Health
- Aggression
- Armor
- Speed
- Weaknesses

---

## 🌍 Dynamic Environments

Fight inside locations such as:

- Subway cars
- Family kitchens
- Damp basements
- Dense greenhouses

Each environment influences encounter outcomes.

---

## ⚔ Tactical Combat

Choose your strategy:

- Fight
- Run
- Hide
- Attempt to tame
- Create completely custom actions

Equipment and emotional stance both affect combat calculations.

---

## 🧠 AI Narrative Engine

When configured with a Gemini API key, the simulator generates cinematic survival scenarios based on:

- Creature
- Environment
- Equipment
- Player stance
- Chosen action

If AI is unavailable, the simulator automatically falls back to a local rule-based engine.

---

## 🖼 AI Image Generation

Generate realistic survival-horror renders of each creature using Google's Imagen model.

Images are generated dynamically from the current encounter.

---

## 🔊 Retro Synth Audio

Original Web Audio synthesizer providing:

- alarms
- impacts
- UI sounds
- warning effects

No external audio assets required.

---

## 📊 Persistent Progress

Tracks:

- Successful extractions
- Casualties
- Mutations discovered

Progress is automatically saved in local browser storage.

---

# Gameplay

1. Select a creature.
2. Choose an environment.
3. Equip a survival tool.
4. Select your behavioral stance.
5. Execute actions.
6. Adapt as the encounter evolves.

Every decision changes the outcome.

---

# AI Integration

The simulator supports two execution modes.

### Online

- Gemini generates dynamic survival narratives.
- Imagen generates realistic creature visuals.

### Offline

If AI services are unavailable, the simulator seamlessly switches to its built-in procedural simulation engine, ensuring the experience remains fully playable.

---

# Design Philosophy

L.A.E.S. combines elements of:

- survival horror
- tactical simulation
- procedural storytelling
- biological speculation
- retro terminal interfaces

The goal is to create encounters that feel unpredictable, cinematic, and entertaining while remaining lightweight enough to run entirely inside a single HTML file.
